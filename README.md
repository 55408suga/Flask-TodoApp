# Portfolio TodoApp

## 概要 (Overview)

REST APIs with Flask and Python in 2025 という Udemy の講座で学んだことをアウトプットするために作成した、Flask での初めてのアプリケーションです。今後様々な機能を実装予定です。

## 使用技術 (Technology Used)

- Python
- Flask
- Flask-Smorest
- Flask-SQLAlchemy
- Marshmallow
- Docker

## 機能一覧 (Feature List)

- Todo の追加
- Todo の一覧表示
- Todo の更新
- Todo の削除

## 実装予定の機能 (Planned Features)

- タグ付け機能
- ユーザー認証
- フロントエンド実装
